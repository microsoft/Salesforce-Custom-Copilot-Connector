# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the [existing
issues](https://github.com/microsoft/Salesforce-Custom-Copilot-Connector/issues) before filing new
issues to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

## Scope of support

This repository is a **template / reference implementation** intended to help customers and
partners build their own Microsoft 365 Copilot custom connectors for Salesforce CRM. It is
provided as-is under the [MIT License](LICENSE), and is **not** a supported Microsoft product.

For help with:

- **Bugs in this template** or **feature requests** — file a GitHub Issue here.
- **Microsoft 365 Copilot custom connector platform questions** — consult the [Graph Connectors documentation](https://learn.microsoft.com/en-us/graph/connecting-external-content-connectors-overview) or contact your Microsoft account team.
- **Production deployments** — engage your Microsoft Customer Success / Partner Success contact. This template is a starting point, not a production-supported deliverable.
- **Salesforce platform questions** — consult [Salesforce Developer documentation](https://developer.salesforce.com/).
- **Security vulnerabilities** — see [SECURITY.md](SECURITY.md). **Do not file a public Issue.**

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
